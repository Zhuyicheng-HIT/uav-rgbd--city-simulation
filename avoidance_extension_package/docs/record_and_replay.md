# 录像与回放说明

录像和回放用于把真实飞行中的 RGB-D 数据、飞控状态、IMU、位姿等信息保存下来，在离线环境中复现算法。

## 录像内容

建议保存：

- RGB 图像。
- 深度图。
- CameraInfo。
- MAVROS state。
- 本地位置。
- IMU / 姿态。
- 安全点输出。
- tracker 目标点。

## 回放用途

- 不起飞即可复现真实飞行中的算法输入。
- 调试 YOLO 检测、深度反投影和坐标转换。
- 对比不同参数下的安全点稳定性。
- 定位实机中无法实时观察的问题。

## 数据不进仓库

`.db3`、rosbag、视频和大日志不进入 GitHub。建议目录中只放 `README.md`：

```text
data/
└── README.md
```

说明数据应该放到哪里，例如：

```text
~/vision_avoid/record_YYYYMMDD_HHMMSS/
```

## 回放注意事项

- 起飞和降落段通常没有行人，调试检测算法时优先使用中间帧。
- CameraInfo 有记录时优先使用真实内参。
- 如果只回放图像，不回放飞控位姿，安全点只能处于体坐标/临时坐标模式，不能代表真实飞行闭环效果。
