# 避障拓展包

本包建立在基础仿真环境之上，提供“识别目标、预测轨迹、生成安全降落点、状态机控制无人机执行避障降落”的完整拓展能力。

## 功能范围

- 仿真视觉节点：订阅 RGB-D 和飞控/仿真位姿，发布安全点。
- 仿真状态机：起飞、悬停、平飞到安全点上方、跟随安全点降落。
- 回放程序：读取真实飞行录制的 `.db3` 和飞控数据，复现视觉算法。
- 实机轻量视觉程序：去除可视化和视频回放，仅保留实时推理、坐标转换、安全点发布、日志。
- 实机状态机：按照 APM 4.6 + D435i 的真实输入输出运行。
- 日志记录：保存安全点、tracker 目标点、无人机坐标、姿态、状态机阶段。
