# 开源前检查清单

## 仓库结构

- [ ] 根目录有 `README.md`，说明两个包的功能和运行入口。
- [ ] `base_sim_package/` 有安装、运行、排坑说明。
- [ ] `avoidance_extension_package/` 有仿真、实机、录像、回放说明。
- [ ] `.gitignore` 已排除大文件、构建产物和个人日志。
- [ ] 模型、录像、数据集没有误提交。

## 安全与隐私

- [ ] 删除 WSL 用户密码、个人 Windows 路径、真实串口号等敏感信息。
- [ ] 删除真实飞行日志中的 GPS 隐私信息，或只发布脱敏样例。
- [ ] 文档中不包含“默认 sudo 密码”等个人环境内容。

## 可复现性

- [ ] 写明 Ubuntu 22.04、ROS2 Humble、Gazebo、MAVROS、ArduPilot 版本。
- [ ] 写明推荐用户名为 `zyc`，或提供完整路径替换说明。
- [ ] 写明 Python 版本和关键依赖版本。
- [ ] 写明相机话题、飞控话题、状态机输入输出。
- [ ] 写明起飞、视觉启动、降落的终端命令顺序。

## 性能

- [ ] YOLO 默认置信度写明为 `0.60`。
- [ ] 图像话题使用 `BEST_EFFORT`，避免视频堆积导致延迟。
- [ ] TensorRT engine 为可选优化，不作为基础运行的强制条件。
- [ ] 说明 `numpy==1.26.4`、`protobuf==3.20.3` 等已踩坑版本约束。

## 实机风险

- [ ] 实机程序与仿真 fallback 分离。
- [ ] 实机状态机保留人工接管能力。
- [ ] 实机降落逻辑在空旷环境逐步测试，不直接在复杂场景中全自动运行。
